# GlobalEvents

Find out about natural events happening around the globe. (Uses NASA's EONET API.)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'global_events'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install global_events

## Usage

To use this gem run

```ruby
ruby bin/run.rb
```
